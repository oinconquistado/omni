# Omni Development Documentation

Welcome to the Omni development documentation.

## Architecture

This is a Turborepo monorepo with:

- **omni-client** (port 3000) - Next.js client application
- **omni-admin** (port 3001) - Next.js admin panel  
- **omni-server** (port 3002) - Fastify API server
- **devdocs** (port 3003) - Development documentation

## Getting Started

```bash
# Install dependencies
pnpm install

# Start all applications
pnpm dev
```
